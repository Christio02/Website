1. Why do mobile phones usually ignore auto play on video elements?

Answer: The reason why mobile phones ignore autoplay on video elements, is because the battery and data usage of the phone will be affected.
It's then better to let the user decide to play the video.

3. Mention some of the advantages of having a single responsive website instead of dedicated versions
like m.til.no for mobile and til.no for desktop?

Answer: The advantages of having a single responsive website instead of dedicated versions are:
- It's easier to maintain, as you only have to maintain one website.
- It's easier to update, because you only have to update the code for one website
- It's easier for users to access the website on any device, as they only need to remember one URL.
