Questions:

  1. What does the server in the client server architecture do?
  2. The server in the client server architecture, is a remote computer that provides service for clients in the architecture.
    This happens because the client requests the data, and the server returns it.
  3. What does the client do?
  4. The client is essentially the other computers that connects to the server to request data from the server.
  5. What is an IP address?
  6. The IP address is the unique identification number which identifies each computer, by using the Internet Protocol (IP)
  7. What is DNS?
  8. A DNS, short for Domain Name System is essentially the dictionary for the internet addresses. It "translates" each
    internet address from (IP) each website and then displays it as readable names.
  9. What is the difference between the Internet and WWW?
  10. The difference between the Internet and WWW, is that the internet is the network consisting of connected computers.
    WWW is the webpages you see on each device.
  11. Explain what HTML is
  12. HTML is the publishing language of the World wide Web, and is a technology used to structure a website.
